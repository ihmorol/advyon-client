# Advyon Color Theme

The Advyon application uses a sophisticated "Deep Teal" color palette, designed to convey professionalism, trust, and modern elegance.

## Color Palette

| Color Name | Hex Code | HSL | Usage |
| :--- | :--- | :--- | :--- |
| **Deepest Teal** | `#051C1B` | `176 70% 6%` | Main Background, Cards, Popovers |
| **Dark Teal** | `#153433` | `178 42% 14%` | Secondary Backgrounds, Muted Elements |
| **Advyon Teal** | `#1C4645` | `178 43% 19%` | Primary Brand Color, Buttons, Headers |
| **Accent Teal** | `#3A7573` | `176 34% 34%` | Borders, Inputs, Interactive States |
| **Soft Teal** | `#B0C4C3` | `176 13% 73%` | Muted Text, Subtitles |
| **White** | `#FFFFFF` | `0 0% 100%` | Primary Text, High Contrast Elements |

## Usage Guidelines

### Backgrounds
- Use `bg-background` (Deepest Teal) for the main page background.
- Use `bg-card` (Deepest Teal) for content containers/cards.
- Use `bg-muted` (Dark Teal) for secondary sections or subtle separation.

### Typography
- Use `text-foreground` (White) for primary headings and body text.
- Use `text-muted-foreground` (Soft Teal) for secondary text, captions, and descriptions.

### Interactive Elements
- **Buttons**: Use `bg-primary` (Advyon Teal) with `text-primary-foreground` (White). Hover states should slightly lighten or darken.
- **Inputs**: Use `border-input` (Accent Teal) for borders. Focus states should use a ring of the same color.

### Borders
- Use `border-border` (Accent Teal) for dividers and card borders to maintain subtle contrast without being harsh.
